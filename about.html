About the project

About This Project

Decade Evolution is an interactive visual storytelling project that explores how cinema has changed from the 1900s to the 2020s.

Using IMDb-style film ranking data, the project visualizes changes in ratings, popularity, genre prevalence, and genre combinations across decades.

What it explores

The project studies genre trends, film ratings, popularity, and the evolution of cinema across different decades.

Data used

The visualizations use IMDb-style film data, including titles, release years, ratings, votes, rankings, directors, runtime, and genres.

How to use it

Users can filter genres, compare decades, hover for details, inspect individual films, and explore how patterns change over time.

Project Goal

The main goal is to make long-term cinema trends understandable through clear, interactive, and visually engaging information visualization.

Instead of only showing a list of films, the project helps users discover historical patterns: which genres became more common, which decades have highly rated films, how popularity is distributed, and which genres tend to appear together.

Visualizations Included

Each visualization answers a different analytical question, so users can move from overview to detail.

Genre Prevalence Heatmap

Shows how common each genre is across decades using a low-to-high color scale.

Average Rating Line Chart

Shows how film ratings change over time and allows comparison by genre.

Rating and Popularity Scatterplot

Shows individual films by release year, rating, and number of IMDb votes.

Decade Comparison Views

Shows how genres increase or decrease between selected decades.

Rating Distribution Chart

Shows the spread of ratings in each decade, not only the average.

Genre Co-occurrence Network

Shows which genres often appear together and how combinations evolve.

Data Meaning

The project treats genres as multi-label categories, meaning one film can belong to several genres. For example, a film can be Adventure, Drama, and Western at the same time.

Genre prevalence is calculated as the percentage of films in a decade that include a given genre. This makes it possible to compare genre importance across decades even when decades contain different numbers of films.

Design Approach

The interface uses a dark cinematic visual style to match the theme of film history. Warm beige and gold tones are used for emphasis, while muted colors reduce visual clutter.

The design prioritizes readability, comparison, visual hierarchy, and meaningful interaction. The objective is to make the project feel cinematic without sacrificing clarity.

Limitations

This project does not claim to represent every film ever made. The results depend on the dataset, the ranking criteria, and the availability of rating and vote information.

Older films may have fewer votes than recent films, which can affect popularity comparisons. Ratings can also be influenced by platform bias, audience demographics, and historical visibility.
